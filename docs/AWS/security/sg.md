

# Security Groups

## Inbound Rules

### Only allow

- Allow only specific IPs
- Allow only specific Ports
- Allow only http or https

### Do not allow

- Open for all like “0.0.0.0/0”
- Port ranges
- unknown ports


## Outbound Rules

### Only allow

- Allow only specific IPs
- Allow only specific Ports
- Allow only http or https

### Do not allow

- Open for all like “0.0.0.0/0”
- Port ranges
- unknown ports
